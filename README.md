# Battery Life Calculator

A simple web calculator for calculating the battery life of an electronic device.

## Features

- Select battery type (CR2032, AAA, AA, 9V Battery) or enter capacity manually
- Enter active time and current
- Enter sleep time and current
- Calculate average current consumption
- Calculate battery life in hours, days, months, years

## Usage

Open `index.html` in a browser. Fill in the fields and click "Calculate".

## Responsive

The page is responsive for mobile devices.
